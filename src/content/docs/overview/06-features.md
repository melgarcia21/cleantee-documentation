# Key Features

## Overview

The CleanTee Online Booking System is built around **three core pillars**: customer convenience, staff empowerment, and administrative efficiency. This page details all major features organized by user role.

## Customer Features

### 1. 3-Step Booking Wizard

**Step 1: Service Selection**

- Browse 8 cleaning service types
- View detailed service descriptions
- See pricing information
- Select service type
- Choose optional add-ons (shampooing)
- Input service area size
- View automatic cleaner calculation

**Step 2: Schedule & Location**

- Select preferred date (24-hour minimum advance)
- Choose time slot (7 AM - 6 PM)
- Enter service address
- Select city from Metro Manila list
- Add postal code
- Include special instructions
- Save location for future use

**Step 3: Review & Payment**

- Review booking summary
- View price breakdown
- Apply loyalty points (optional)
- Select payment method (GCash or Cash)
- Agree to terms and conditions
- Confirm and submit booking
- Receive instant confirmation

### 2. Automatic Cleaner Calculation

**How It Works:**

- Input service area in square meters
- System calculates: Area ÷ 50 sqm = Required cleaners
- Adjustment for multiple floors
- Display estimated service duration
- Show total personnel assigned
- Transparent calculation logic

**Example:**

```
Area: 150 sqm
Floors: 2
Calculation: 150 ÷ 50 = 3 cleaners
Adjustment: +1 cleaner for multiple floors
Total: 4 cleaners assigned
```
